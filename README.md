## Testing Environment Variables

The following environment variables must be set (and valid) for tests to run.

| Key            | Description         |
| :------------- | :------------------ |
| `NFA_URL`      | URL to NFA instance |
| `NFA_USERNAME` | NFA Username        |
| `NFA_PASSWORD` | NFA Password        |
